POKRETANJE PROJEKTA (WINDOWS):
1. Build src foldera u Cmake i pokretanje u Visual Studio (označiti tgti kao "Startup Project")
2. Sa lokacije C:\Users\User\natID.RAMDisk\Out\TGTIPluginSol kopirati .dll file
3. Na lokaciju C:\Program Files\IDz\Digital Twin\res\plugins zalijepiti .dll file kopiran u prethodnom koraku
4. Otvoriti dTwin
5. model -> import -> Turbine Governor Type I
